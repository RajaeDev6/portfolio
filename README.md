# Rajae - Personal Portfolio

A clean and minimal portfolio website that highlights my software engineering and cybersecurity work.  
The site focuses on clarity, structure, and simple navigation while showcasing both my engineering projects and academic cybersecurity labs.

Live Demo: [Rajae's portfolio](https://rajaedev.netlify.app)

---

## Tech Used

- **HTML**
- **TailwindCSS**
- **JavaScript**
- **FontAwesome Icons**

No frameworks or build tools - the site is lightweight and loads quickly by design.

---

## Features

### **1. Responsive Layout**
Works smoothly across desktop, tablet, and mobile.  
Includes a clean mobile hamburger menu with auto-close behavior.

### **2. Projects Section**
Showcases software engineering work:
- Public Eye JA (AI infrastructure reporting)
- Assistia (AI learning assistant)

- PingSweeper (Go subnet scanner)

### **3. Cybersecurity Projects**
Summaries of academic labs and research:

### **4. Smooth Fade Animations**
Sections fade in naturally as the user scrolls.

### **5. Minimalist Aesthetic**
Thin typography, soft grays, and subtle borders inspired by modern interface design.

## The Process

The goal was to build a portfolio that reflects how I approach software:  
simple, functional, and intentional.

1. **Design First**  
   Planned a clean layout with clear sections and consistent spacing.  
   Prioritized readability over decoration.

2. **TailwindCSS**  
   Used Tailwind for utility-based styling and fast iteration.  
   Kept the design minimal - no custom components or heavy styling.

3. **Mobile Experience**  
   Focused heavily on mobile:
   - hamburger menu  
   - auto-closing navigation  
   - smooth scrolling  
   - clean spacing  

4. **Content Structure**  
   Split the work into:
   - About  
   - Software Projects  
   - Cybersecurity Projects  
   - Skills  
   - Certifications  
   - Contact  

   This keeps the portfolio balanced between software engineering and cybersecurity.

5. **Performance**  
   Avoided frameworks and build pipelines to keep things simple and load times extremely fast.
